# CSCT Cloud
CSCT Cloud is a cloud server used for teaching and learning activities in the [School of Computer Science and Creative Technologies (SCCT)](https://www.uwe.ac.uk/about/colleges-and-schools/arts-technology-and-environment/computing-creative-technologies){:target="_blank"} at [UWE Bristol](https://www.uwe.ac.uk/){:target="_blank"}.

It runs Ubuntu (24.04), is hosted using [Azure](http://azure.microsoft.com/){:target="_blank"}, and provides students with access to a Linux-based system with the necessary software installed to complete work.

## Who has access to the server?
Students taking a module that uses the server will automatically be given access - in practice this is normally all students on the Computer Science and Cyber Security programmes.

If you don't have access to the server and think you should, talk to your module tutors - sometimes modules that should have access need to be re-added by IT Services.

## Why is it called CSCT Cloud and not SCCT Cloud?
The server was created before the old Department (CSCT) became the current School (SCCT) and it hasn't been renamed yet!
