    Choosing a terminal

    You'll need to use a terminal program to interact with CSCT Cloud, all operating systems come with at least one, there are also other third party terminal terminal applications you can use.

    Windows

    Windows includes two terminal programs by default - PowerShell and Command Prompt - it's recommended you use PowerShell, you can launch this from the start menu (press the Win key and search for it).

    Microsoft offer Windows Terminal which acts as a wrapper around other terminal programs on your computer, offering useful features like tabs and panes and better text customisation.

    It is also possible to run a Linux environment on your Windows machine, using Windows Sybsystem for Linux - supporting you with the installation and setup of this is beyond the scope of this guide however.

    macOS

    macOS comes with Terminal, you can launch this from Spotlight - use Cmd+Space and search.

    iTerm2 and Hyper are good terminal programs, but there are many alternatives available.

    Linux

    Your distribution will come with at least one terminal program, often just called terminal, but will possibly include several programs - pick one!

    There are many alternatives available.
